Esenciales de Javascript

  • Javascript es un lenguaje de programacion enfocado en el desarrollo de paginas web.
  • Las se declaran variables y se utilizan para capturar datos modificables.
  • Let nos permite declarar variables que enfrascadas en el bloque de codigo en el que son declaradas.
  • Const declara constantes que no se pueden reasignar, se utiliza para constantes que peramenecen permanentemente.

    • Funciones

  • Las funciones contienen un bloque de codigo que se corre al momento en el que se llame la funcion dentro del bloque de htmml utilizando funcion()
  • Las funciones pueden tener parametros que se utilizan al llamar la funcion con funcion(parametro1, parametro2...etc)

    • innerHTML y getElementById

  • getElementById es uno entre muchos apuntadores que agarran elementos dentro del html dependiendo de su nombre y tipo de elemento
  • innerHTML se utiliza para sobreescribir el contenido de cualquier elemento dentro del HTML
  • document. se refiere al documento HTML en el que se esta trabajando, se utliza document.getElementByID("").innerHTML="" para sobreescribir un elemento
